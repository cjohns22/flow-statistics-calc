How to use this program:

  1. Ensure that all files are located in the same folder. Create input and output *.CSV files. The input file should have dates in the first column and streamflow in the last column. The output file should be blank.
  2. Download the streamflow parser script to your folder. This script will model undocumented flows between days such that the difference between days is no more that a week.
  3. In the terminal, go to your project folder and run the following code: python streamflow_parser_script.py your_input.csv your_output.csv
  4. Select your_output.csv (below) that you specified in step 3. This is the file used to generate statistical information.
  5. Refresh this page to start over.
